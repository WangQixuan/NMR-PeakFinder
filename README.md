# NMR-PeakFinder

## 项目简介
NMR-PeakFinder 是一个专为化学研究人员设计的工具，用于查询常见实验室溶剂作为微量杂质的核磁共振化学位移。通过简单的界面，用户可以快速找到特定化学位移范围内可能对应的化合物峰，提高 NMR 谱图解析效率。

## 主要功能

- **双模式查询**：支持 ¹H-NMR 和 ¹³C-NMR 谱图数据查询
- **多种溶剂支持**：内置 7 种常用氘代溶剂的化学位移数据
- **精确搜索**：可设置化学位移值和阈值范围进行精准查询
- **搜索历史记录**：自动保存最近 10 次查询记录，便于重复操作
- **响应式设计**：适配桌面和移动设备，随时随地使用
- **美观界面**：现代化 UI 设计，流畅的动画效果，提升用户体验

## 支持的溶剂

- CDCl₃ (氯仿-d)
- (CD₃)₂CO (丙酮-d₆)
- (CD₃)₂SO (二甲亚砜-d₆)
- C₆D₆ (苯-d₆)
- CD₃CN (乙腈-d₃)
- CD₃OD (甲醇-d₄)
- D₂O (重水)

## 使用方法

1. **选择核磁共振类型**：从下拉菜单中选择 ¹H-NMR 或 ¹³C-NMR
2. **选择溶剂**：选择您实验中使用的氘代溶剂
3. **输入化学位移值**：在化学位移输入框中输入目标值（单位：ppm）
4. **设置阈值范围**：输入阈值以定义搜索范围（例如：0.10 表示查询 ±0.10 ppm 范围内的峰）
5. **点击查询**：获取匹配结果
6. **查看历史记录**：通过搜索历史按钮可快速恢复之前的查询参数

## 结果解读

- **高亮行**：绿色高亮显示的行表示完全匹配您查询条件的峰值
- **化合物信息**：显示化合物名称、质子/碳原子类型、多重峰信息和化学位移值
- **无结果提示**：当未找到匹配结果时，会显示友好提示信息

## 技术实现

本项目使用纯前端技术构建：

- HTML5 语义化标签
- CSS3 实现响应式设计和动画效果
- JavaScript/jQuery 处理交互逻辑
- Bootstrap 提供基础 UI 组件
- LocalStorage 存储搜索历史
